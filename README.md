# HappyCapy 技能目录 · 中文版

> 收录 HappyCapy 平台 4,221 个 AI 沙箱技能，全部翻译为中文，支持分类浏览与搜索，一键跳转安装。

**在线访问：** 直接打开 `happycapy-skills-zh.html` 即可使用，无需服务器，无需登录。

---

## 功能特性

- **4,221 个技能** — 来自 skillsmp 平台，涵盖官方与社区技能
- **全中文翻译** — 技能名称与描述均已翻译，英文原名保留为小字
- **五大分类** — 效率工具 / 开发编程 / AI&数据 / 系统运维 / 内容创作
- **关键词搜索** — 实时过滤，支持中英文混搜
- **按使用率排序** — 默认按星标数（使用量）降序展示
- **官方技能标注** — 来自 OpenClaw / Anthropic / Vercel 等官方作者单独标注
- **一键安装** — "+ 安装" 按钮直链 skillsmp.com 安装页
- **PWA 支持** — 可添加到手机桌面，Service Worker 缓存加速
- **深色模式** — 跟随系统自动切换
- **全设备适配** — 手机 / 平板 / 桌面全尺寸响应式布局

## 使用方式

直接下载 `happycapy-skills-zh.html`，在浏览器中打开即可。
